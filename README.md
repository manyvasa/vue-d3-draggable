# Visualization of user groups.
#### Vue + Vuex + d3(d3drag, d3zoom) + VueDraggable
You can use:
* add group
* delete group
* move users
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
